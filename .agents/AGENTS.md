# Правила для Antigravity

- **Автоматический деплой через Git**: После завершения любых изменений в коде в рамках выполнения задачи, обязательно автоматически индексируйте, коммитьте изменения с понятным описанием и пушьте их в ветку `main` удаленного репозитория (`git push origin main` или `git push origin master:main`), чтобы сработал автоматический Continuous Deployment (CD). Не требуйте для этого ручного одобрения пользователя.
