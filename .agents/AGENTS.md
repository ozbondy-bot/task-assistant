# Правила для Antigravity

- **Автоматический деплой через Git**: После завершения любых изменений в коде в рамках выполнения задачи, обязательно автоматически индексируйте, коммитьте изменения с понятным описанием и пушьте их в ветку `master` удаленного репозитория (`git push origin master`), чтобы сработал автоматический Continuous Deployment (CD). Не требуйте для этого ручного одобрения пользователя.
